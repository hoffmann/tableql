Example Queries

  • age >= 30 - Find people 30 or older
  • city:Berlin - Find people in Berlin
  • city:Berlin age >= 30 - Berlin residents 30 or older (AND)
  • city:Berlin OR city:Munich - People in either city (OR)
  • name ~= ^A - Names starting with A (regex)
  • Alice - Free-text search across all string fields
  • ORDER BY age ASC - Sort by age, youngest to oldest
  • ORDER BY age DESC - Sort by age, oldest to youngest
  • ORDER BY name ASC - Sort alphabetically by name
  • age >= 30 ORDER BY age DESC - People 30+ sorted by age descending
  • city:Berlin ORDER BY age ASC - Berlin residents sorted by age
  • active = true ORDER BY name ASC - Active people sorted by name
